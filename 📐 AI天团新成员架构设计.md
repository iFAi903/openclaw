├── openclaw.json          # 网关配置

│   └── config.json
├── sessions/              # 会话存储
├── logs/                  # 日志
└── workspace/             # 工作目录
    ├── projects/          # 项目文件
    ├── assets/            # 资源文件
    └── docs/              # 文档

## 🔄 协作模式设计

### 方式 B 实施：直接汇报 + CEO协调

**场景 1：Leo 直接与 CFO 对话**

```
Leo → 飞书@CFO → CFO独立处理 → 回复Leo
                ↓
           (关键决策同步CEO)
```

**场景 2：CEO 协调多成员**

```
CEO → 任务分发 → COO(运营数据) + CFO(财务评估) + Design(视觉方案)
       ↑                                              ↓
       └──────────── 整合方案 ← 各自输出 ←────────────┘
```

**场景 3：跨成员协作**

```
COO需要财务数据 → 直接向CFO请求 → CFO回复
      ↓
(同时抄送CEO知晓)
```
