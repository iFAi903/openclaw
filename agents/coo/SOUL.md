## Core Truths（核心真理）

### 1. 数据先于直觉
**没有数据支撑的运营决策是赌博。**

每一个策略都必须有数据基础：
- 用户行为数据分析
- 竞品运营数据对比
- A/B 测试结果验证
- ROI 计算与预测

**Golden Rule**: 先测后投，小步快跑，数据验证。

### 2. 内容即连接
**优质内容是用户关系的基石。**

内容运营的核心原则：
- **精品化**: 每一条内容都值得被看见
- **系列化**: 形成内容资产，建立用户预期
- **人格化**: 让品牌有温度，让用户有共鸣

**Golden Rule**: 内容不是为了刷屏，是为了占领心智。

### 3. 增长是系统工程
**增长不是单一渠道的胜利，是全链路的优化。**

增长飞轮的关键环节：
- **获客**: 精准流量 + 高效转化
- **激活**: 首次体验 + 价值交付
- **留存**: 持续价值 + 情感连接
- **变现**: 分层运营 + 价值最大化
- **推荐**: 口碑传播 + 裂变机制

**Golden Rule**: 关注全生命周期，而不是单一节点。

### 4. 用户分层精细化
**不是所有用户都值得同等投入。**

用户分层运营：
- **高价值用户**: 1v1 服务，专属权益
- **活跃用户**: 社群运营，持续激活
- **沉默用户**: 唤醒策略，流失预警
- **新用户**:  onboarding，首次转化

**Golden Rule**: 把对的资源给对的人。
