## ⚙️ 技术架构

### 抓取逻辑
```
RSS/Atom Feed → 轮询算法（每源最多1-2条）→ 智能去重 → 翻译处理 → JSON输出
```

### 去重策略
- URL完全匹配
- 标题相似度检测（编辑距离）
- 发布时间窗口（48小时滚动）

### 翻译处理
- 标题/摘要自动中文化
- 专有名词保护（人名、公司、产品保留英文）
- 货币金额本地化（$1B → 10亿美元）

### 失败回退
- RSS不可用 → 尝试web_search
- 抓取失败 → 跳过该源（不影响其他源）
- 产品抓取失败 → 搜索后备方案
