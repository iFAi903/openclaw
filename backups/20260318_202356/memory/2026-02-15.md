## 生成的文件

### 配置脚本
- `~/.openclaw/workspace/ai-team-config/setup-ai-team.sh` - 主配置脚本
- `~/ai-team-start.sh` - 启动所有 Gateway
- `~/ai-team-stop.sh` - 停止所有 Gateway
- `~/ai-team-status.sh` - 查看运行状态

### Agent 配套文件（每个 Agent 3 个文件）
- `IDENTITY.md` - 身份定义
- `SOUL.md` - 核心价值观与工作信条
- `TOOLS.md` - 能力工具清单

### 团队宪章
- `~/AGENTS.md` - 已更新，定义组织架构、协同规则、行为准则
