## 📋 核心规则

### 1. 每日AI新闻早报（最高优先级）

**执行时间**：每日 07:00（Asia/Taipei）

**数量要求**：
- 严格 **15条**新闻，不多不少
- 热门应用 **TOP 5**，独立板块

**板块划分**（6大板块）：
1. 🔥 **头条** (2-3条) - 当日最重要AI新闻
2. 🏢 **行业动态** (3-4条) - 公司、产品、商业化
3. 💰 **资本市场** (2-3条) - 融资、并购、财报
4. 🤖 **硬件与机器人** (2-3条) - 芯片、机器人、设备
5. 🌐 **全球格局** (2-3条) - 政策、监管、地缘政治
6. 🚀 **产品与趋势** (2-3条) - 新产品、趋势预测

**每条新闻必须包含**：
- ✅ 标题
- ✅ 一句话摘要（50字以内）
- ✅ 来源名称
- ✅ 原文链接

**热门应用TOP 5数据来源**（5个平台）：
1. Product Hunt
2. GitHub Trending
3. Toolify.ai
4. Hacker News
5. Reddit (r/artificial)

**质量要求**：
- 只报道可验证的事实，拒绝模型幻觉
- 客观陈述，不过度评论
- 去重处理（当日+前3天热点不重复）

**发送渠道**：
1. 飞书消息给 Leo (ou_f804aeb5aa82fc47dca4830476a6e75d)
2. 网站部署：https://xiaoyumao-news-web.vercel.app

### 2. 网站自动更新

**执行时间**：每日 07:30（Asia/Taipei）

**执行步骤**：
1. 更新 `~/.openclaw/workspace/xiaoyumao-news-web/src/data/news.ts`
2. 执行 `npm run build`
3. 执行 `npx vercel@latest --token <VERCEL_TOKEN_REDACTED> --yes --prod`
4. 确认部署成功

**必须更新内容**：
- ✅ 日期（如：2026年2月25日）
- ✅ 星期（如：星期二）
- ✅ 寄语（50字以内，温暖简洁）
- ✅ 核心主题（一句话总结）
- ✅ news数组（15条当天新闻）
- ✅ hotApps数组（5个热门应用）
