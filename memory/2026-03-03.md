# 2026-03-03 - 小羽毛 AI 新闻早报系统重建

## 今日核心任务
重建 AI 新闻早报系统，采用 Pi-Native 架构，由 CEO 统筹，CTO 执行技术实现。

## 系统架构
- **数据层**: Python RSS 抓取脚本 (`fetch_news_v2.py`)
  - 国际源: The Verge, TechCrunch, Wired, VentureBeat, Reddit
  - 中国源: 极客公园 (GeekPark) - 因 RSS 超时采用搜索回退策略
  - 产品源: Product Hunt, Hacker News, GitHub, Toolify
  - 去重机制: 3 天滚动历史记录 (news_history.json)
  
- **前端层**: Next.js 16 + Tailwind CSS
  - 四色系统设计: 高级灰(行业)、蓝渐变(头条)、深蓝(总结)、紫色(资本)
  - 图标系统: Lucide React
  - 响应式布局，支持移动端

- **部署层**: Vercel

## 关键技术决策
1. **RSS 抓取**: 使用 curl + User-Agent 模拟浏览器，避免基础反爬
2. **数据回退**: GeekPark RSS 不可用时，采用 web_search 获取最新内容
3. **类型安全**: TypeScript 接口定义 (DailyNews, NewsItem, AppItem)
4. **静态生成**: Next.js SSG 模式，构建时预渲染所有页面
