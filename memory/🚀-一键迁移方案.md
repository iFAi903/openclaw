# 🚀 印象笔记 → Obsidian 一键迁移方案

## ⚡ 超简流程（只需 30 秒人工操作）

### 第 1 步：一键导出（30秒）
1. 打开印象笔记
2. 按 `Command + A` 全选所有笔记
3. 菜单：`文件` → `导出笔记...`
4. 格式选 `ENEX` → 保存到桌面
5. **完成！剩下的全交给我**
